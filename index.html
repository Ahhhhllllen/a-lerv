Caesar Cipher

The Caesar cipher is a substitution cipher where each letter in the plaintext is shifted a certain number of places down or up the alphabet. For example, with a shift of 3, A would be replaced by D, B would become E, and so on.

Atbash Cipher

The Atbash cipher is a substitution cipher where each letter in the plaintext is replaced by its reverse in the alphabet. For example, A becomes Z, B becomes Y, and so on.

ROT13 Cipher

The ROT13 cipher is a special case of the Caesar cipher where each letter is shifted 13 places. It is its own inverse, meaning applying ROT13 twice will return the original text.

Base64 Encoding

Base64 is a method of encoding binary data into an ASCII string format. It is commonly used to encode data for transmission over text-based protocols, such as email attachments or storing data in URLs.
